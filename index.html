Architecture Explanation

1
Start CLI App: The application launches via command line interface.
2
Read Code File: The app reads the specified code source file.
3
Extract Method for Testing: The app identifies and extracts the specific method(s) that need unit testing.
4
Preprocess Method Code: The extracted code is formatted and prepared for model input.
5
Send to Local Model: The preprocessed code is sent to a locally running model.
6
Receive Initial Unit Test: The local model generates a basic unit test for the method.
7
Enhance Test with Public Large Model: The initial unit test is sent to a public large language model with instructions to enhance and professionalize it.
8
Receive Enhanced Professional Test: The public model returns an improved, professional-quality unit test.
9
Generate Final Unit Test File: The app formats the enhanced test into a proper unit test file.
10
Save to Project Directory: The final unit test is saved in the appropriate project directory.
